# 隱私權政策 — 司法院裁判書閱讀助手

**最後更新：2026-04-20**
**擴充功能版本：v0.2.6**

---

## 📌 摘要

本擴充功能**不向任何遠端伺服器發送請求**、**開發者完全無法接觸使用者之任何資料**、**不追蹤、不分析、不投放廣告、不轉售**。所有運算均於使用者本機瀏覽器內完成；側邊欄顯示之剪貼簿紀錄僅存於記憶體，關閉瀏覽器即自動清空。

---

## ✅ 本擴充功能**技術上無法為之**之事項

以下行為於設計上**本擴充功能技術上即無法執行**，非僅政策層面之承諾：

- ❌ 不向任何遠端伺服器發送 HTTP／HTTPS／WebSocket／fetch 請求
- ❌ 不使用任何第三方分析服務（Google Analytics、Sentry、Mixpanel 等均無）
- ❌ 不嵌入任何廣告相關套件
- ❌ 不發送遙測資料、錯誤回報或使用量指標
- ❌ 不將使用者資料傳送、轉售或授權予任何第三方
- ❌ 不將使用者資料用於信用評估、放款決策或其他與本擴充功能單一用途無關之目的
- ❌ 不讀取、儲存或上傳其他瀏覽紀錄、書籤、Cookie、密碼或帳號資訊
- ❌ 不於司法院裁判書系統以外之任何網站運作
- ❌ 不使用 `chrome.identity`、`webRequest`、`cookies`、`history`、`bookmarks`、`activeTab`、`clipboardRead`、`scripting`、`downloads` 等高風險 API

上開各項均可透過檢視原始碼（詳文末連結）自行驗證。

## 📂 本機儲存之資料清單

本擴充功能確實會於**使用者本機之瀏覽器程序內**暫存若干資料，惟**所有資料皆僅存於使用者本機，開發者完全無從接觸**：

| 儲存位置 | 內容 | 資料量級 | 生命週期 | 開發者可否存取 |
|---|---|---|---|---|
| `chrome.storage.session` | 判決剪貼簿卡片（當次瀏覽期間複製之文字、裁判字號、時間戳記、原始網址） | 數 KB 至數百 KB | **僅存於記憶體**；關閉瀏覽器即自動清空；不寫入硬碟 | ❌ 否。資料從不離開使用者本機之瀏覽器程序 |
| `chrome.storage.sync` | 使用者偏好設定（複製附字號開關、參照耳標開關、耳標左右位置、展開深度） | < 1 KB | 跨裝置同步予**使用者本人**之 Google 帳號，由 Chrome 加密同步 | ❌ 否。同步由 Google Chrome 基礎設施管理，開發者無 API 可讀取任何使用者之同步內容 |
| `chrome.storage.local` | 側邊欄字體大小偏好（單一整數） | 數 bytes | 保存於本機 Chrome 使用者設定檔，不跨裝置同步 | ❌ 否。僅存於使用者本機磁碟 |

### 關於 `chrome.storage.session` 之補充說明

`chrome.storage.session` 為 Chrome 於 Manifest V3 引入之**原生記憶體儲存區**，技術特性如下：

- **完全不寫入硬碟**，僅存於 Chrome 程序之記憶體
- 關閉瀏覽器、登出帳戶、Chrome 異常中止或重新啟動時自動清空
- **不透過 `chrome.storage.sync` 同步**，不會傳遞至其他裝置
- 擴充功能之不同執行環境（content script、service worker、側邊欄）共享同一份資料，惟**資料永不離開該瀏覽器使用者設定檔**

此為判決剪貼簿「關閉瀏覽器即清空」行為之底層保證，非政策層面之承諾，而係技術層面不可能為之。

## 🔍 擴充功能的實際行為（逐步說明）

1. **僅在下列網域載入**：
   - `https://legal.judicial.gov.tw/FINT/*` — 司法院法令判解系統
   - `https://judgment.judicial.gov.tw/FJUD/*` — 司法院裁判書系統
   - `https://legal.law.intraj/FINT/*` — 司法院內部網路判解函釋（限院內環境）
   - `https://judgment.law.intraj/FJUD/*` — 司法院內部網路裁判書系統（限院內環境）

   其他任何網站**均不會載入**本擴充功能之程式碼——瀏覽器僅會將 content script 注入至上開四組網址比對規則之頁面，其他一概不注入。

2. **讀取當前判決頁面之 DOM 結構**：以 JavaScript 分析頁面文字節點，偵測判決書常見之階層編號（壹／一／(一)／1.／(1)／①）以產生左側「判決架構」導覽。所有分析均於瀏覽器 JavaScript 引擎內完成，**任何文字片段均不會離開使用者裝置**。

3. **解析正文引用之權威意旨**：於同一 DOM 文字上以正規表示式偵測最高法院判決／裁定、最高法院大法庭裁定、最高法院民刑事庭會議決議、法律座談會決議、大法官釋字解釋、憲法法庭判決及最高行政法院庭長法官聯席會議決議等七類引用，彙整為側邊欄「參照」頁籤供檢視。此一分析全程於本機記憶體進行，不產生任何網路請求，亦不寫入永久儲存。

4. **攔截複製事件**（`copy` 事件）：於上開網域選取文字並按 Cmd+C／Ctrl+C 時，本擴充功能執行下列步驟：
   - 移除選取文字之換行與中日韓字元間之襯邊空白
   - 於尾端附上「（＜裁判字號＞意旨參照）」
   - 寫回使用者之系統剪貼簿
   - 將完整文字加入判決剪貼簿側邊欄（存於 `chrome.storage.session` 記憶體，關閉瀏覽器即清空）

   **文字僅於使用者之瀏覽器、剪貼簿與記憶體之間流動，絕不傳送至網路**。

5. **判決剪貼簿側邊欄**：以 Chrome 原生 `sidePanel` API 呈現卡片列表。點擊卡片上之「複製」鈕，將透過 `navigator.clipboard.writeText()` 把卡片內容寫回系統剪貼簿，供使用者於其他分頁（Google Docs、Obsidian 等）貼上。匯出之 `.txt`／`.md` 檔案由**瀏覽器本身**直接下載至本機磁碟——本擴充功能未取得 `downloads` 權限，無法主動存取檔案系統，純屬瀏覽器之原生下載行為。

### 關於「網頁內容」之詳細說明

為產生判決架構導覽、參照頁籤與智慧複製功能，本擴充功能**必須**以 JavaScript 讀取當前裁判書頁面之文字內容（亦為一切 content script 類擴充功能之共同特性）。該等文字：

1. 僅於**使用者瀏覽器之 JavaScript 引擎記憶體內**處理
2. 從不透過 fetch、XHR、WebSocket 或其他任何方式傳送至遠端
3. **複製**時同步寫入 `chrome.storage.session`（記憶體暫存），關閉瀏覽器即消失
4. 開發者與任何第三方皆**無任何管道**可存取該等文字

依 Chrome Web Store 開發人員計畫政策，資料若**僅於使用者本機裝置暫時處理、從未離開裝置**，不構成「資料收集」。本擴充功能完全符合此條件。

## 🔑 已取得之權限與用途

| 權限 | 用途說明 |
|---|---|
| `host: legal.judicial.gov.tw/FINT/*` | 於法令判解系統之裁判書頁面注入判決架構側欄、攔截複製事件 |
| `host: judgment.judicial.gov.tw/FJUD/*` | 於裁判書系統之裁判書頁面注入判決架構側欄、攔截複製事件 |
| `host: legal.law.intraj/FINT/*` | 同上，適用於司法院內部網路判解函釋 |
| `host: judgment.law.intraj/FJUD/*` | 同上，適用於司法院內部網路裁判書系統 |
| `storage` | 儲存使用者偏好設定（sync）、判決剪貼簿卡片（session 記憶體）、字體大小偏好（local）——全數為本機儲存，無任何遠端傳輸 |
| `sidePanel` | 提供「判決剪貼簿」原生側邊欄介面，以利跨分頁檢視及再次複製當次瀏覽紀錄 |
| `tabs` | 供「前往原文段落」功能使用：查詢已開啟分頁之網址，以判斷原判決頁是否仍存在、決定切換或新開分頁。**僅比對上表所列司法院網域之網址**，不存取分頁內容、不記錄瀏覽歷史、不發送任何網路請求 |

**明確聲明未使用**之敏感 API：`activeTab`、`cookies`、`history`、`bookmarks`、`webRequest`、`webRequestBlocking`、`declarativeNetRequest`、`identity`、`downloads`、`clipboardRead`、`scripting`、`nativeMessaging`、`management`、`geolocation`、`notifications`。
